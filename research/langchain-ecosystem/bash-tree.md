# Bash commands for langchain-ai.docs

## Generate XML tree

```bash
tree -dX -o research/langchain-ecosystem/langchain-ai-docs.xml repos/langchain-ai-docs
```

## Concept

* Linux 系統的檔案系統命名"適合分段執行"
* 適合查閱字典式的檢索
* 一般人看不懂的設計，表示一定有特殊用途

## 最終目標

* 調查 tree 指令的選項設計緣由，擴展至其他指令的選項設計
* 完成 langchain ecosystem 的檔案結構分析。
* 完成 langchain ecosystem 對本專案 `arkhon-rheo` 是否提供更佳的解決方案。
